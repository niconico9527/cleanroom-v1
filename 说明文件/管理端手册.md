# 管理端操作手册

**访问地址**：`admin.html`（或在线地址 https://cleanroom1.netlify.app/admin.html）

管理端用来设置各洁净等级下每个检测项目的单价，设置完后发布给业务员使用。

## 1. 数据同步区

页面顶部有两个卡片：

**云端发布与同步区**（左侧）：
- **拉取云端数据**：从 Gitee 仓库下载最新的价格配置，覆盖当前浏览器里的数据。适用于换电脑、或想撤回最近的修改。
- **发布至云端**：把当前配置推送到 Gitee 仓库。推送后，业务员端会在下次轮询时自动获取更新。
- 下方状态栏显示当前数据来源和版本时间。绿色表示有云端数据，橙色表示仅有本地数据。

**离线配置包管理区**（右侧）：
- **导入本地数据**：从 `.crm` 文件加载价格配置。
- **导出数据到本地**：把当前配置导出为 `.crm` 文件（文件名格式：`20260228单价同步包.crm`），可以线下发给业务员。

## 2. 单价配置

1. 先从"配置目标级别"下拉框选择一个等级（如"10000级"）。
2. 下方表格列出该等级下所有 14 个检测项目，每项有三列可填：
   - **阈值点数**：阶梯计价的分界线
   - **阈值内单价**：低于等于阈值时的单价（元/点）
   - **超阈值单价**：超过阈值时的单价（元/点）
3. 输入后自动保存到浏览器本地，不需要手动点保存。
4. 每列表头有"向下填充"按钮：把该列第一行的数值复制到该列所有行。

## 3. 跨级别复制

如果多个等级的定价方案相同，不需要逐个配置：

1. 先选好一个等级并配置好价格。
2. 点击"复制当前配置到其它级别..."按钮。
3. 在弹窗中勾选要复制到的目标等级（可按分组全选）。
4. 点击"确认应用"，目标等级的配置会被覆盖为当前等级的配置。

## 4. 发布流程

配置完成并确认无误后：
- 有网络：点击"发布至云端"，业务员端会自动同步。
- 无网络：点击"导出数据到本地"，把 `.crm` 文件发给业务员。
